### Card (variant:'default')

Card extends the [Card](https://theme-ui.com/components/card) provided by Theme UI

```js
  <Card />
```

### Variants of Card

```js
  <Card variant="default" />
  <Card variant="primary" />
  <Card variant="secondary" />
  <Card variant="outline" />
  <Card variant="accent" />
  <Card variant="muted" />
```
