### Embed (variant:'default')

Embed extends the [Embed](https://theme-ui.com/components/embed) provided by Theme UI

```js
  <Embed />
```

### Variants of Embed

```js
  <Embed variant="default" />
  <Embed variant="primary" />
  <Embed variant="secondary" />
  <Embed variant="outline" />
  <Embed variant="accent" />
  <Embed variant="muted" />
```
