### Input (variant:'default')

Input extends the [Input](https://theme-ui.com/components/input) provided by Theme UI

```js
  <Input />
```

### Variants of Input

```js
  <Input variant="default" />
  <Input variant="primary" />
  <Input variant="secondary" />
  <Input variant="outline" />
  <Input variant="accent" />
  <Input variant="muted" />
```
