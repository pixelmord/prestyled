### Radio (variant:'default')

Radio extends the [Radio](https://theme-ui.com/components/radio) provided by Theme UI

```js
  <Radio />
```

### Variants of Radio

```js
  <Radio variant="default" />
  <Radio variant="primary" />
  <Radio variant="secondary" />
  <Radio variant="outline" />
  <Radio variant="accent" />
  <Radio variant="muted" />
```
